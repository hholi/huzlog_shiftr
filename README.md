# ESP8266 logger and monitor #

Simple environment sense, message pub/sub and monitoring.

Using ESP8266, HDC1000, OLED.
Shiftr.io as MQTT message broker.


### User stories

* Running on battery
* Deep sleep
* Small case for sensor module
